Replace any non-DesignTokens spacing with the appropriate token:
- 4px → `DesignTokens.spaceXs`
- 8px → `DesignTokens.spaceSm`

- [ ] **Step 2: Vertex handles**
Draw vertex handles as part of `ZoneMapPainter` (currently they're only for hit-test). Add a paint pass that draws circles on selected zone vertices:
```dart
// In ZoneMapPainter.paint(), after drawing zones:
if (selectedZoneId != null) {
  final selected = zones.firstWhereOrNull((z) => z.id == selectedZoneId);
  if (selected != null) {
    final pts = _getPoints(selected, size);
    final handlePaint = Paint()
      ..color = Color(selected.colorValue)
      ..style = PaintingStyle.fill;
    final borderPaint = Paint()
      ..color = Colors.white
      ..style = PaintingStyle.stroke
      ..strokeWidth = 2;
    for (final p in pts) {
      canvas.drawCircle(p, 7, handlePaint);
      canvas.drawCircle(p, 7, borderPaint);
    }
  }
}
```
